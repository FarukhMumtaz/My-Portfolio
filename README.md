## 🚀 Featured Projects

### 🛡️ [Aegis-Flow](https://github.com/FarukhMumtaz/Aegis-Flow.git)
A containerized backend powerhouse featuring custom authentication middleware and PostgreSQL persistence. Optimized for DevSecOps workflows and high-performance security.
* **Tech:** Docker, Node.js, PostgreSQL, JWT.

### 🤖 [AWS Security Guardian](https://github.com/FarukhMumtaz/AWS-Security-Guardian-Bot.git)
An automated IAM threat remediation bot. It monitors CloudTrail logs via EventBridge and triggers Lambda functions to neutralize unauthorized IAM changes in real-time.
* **Tech:** Python (Boto3), AWS Lambda, EventBridge, CloudTrail.

### 🔌 [LLM Security Gateway](https://github.com/FarukhMumtaz/LLM-Security-Gateway.git)
A modular middleware designed to protect Large Language Models. Features include prompt injection detection and PII anonymization to ensure secure AI deployments.
* **Tech:** Python, OpenAI API, Regex Engine, Policy-as-Code.

### 📊 [MBPLDS - DSA Project](https://github.com/hamzaali-712/MBPLDS-DSA-Project-.git)
A complex Data Structures & Algorithms implementation focused on efficient data management and search optimization.
